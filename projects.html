PacMen Exercise

The Pacmen Exercise was the first project I ever made with JavaScript & HTML. This program has an easy to view color scheme and contains two buttons. The first button unilizes a factory that produces a pacman with evry click. The second button sets all existing and future pacmen into random motion. This project is fairly simple but I chose to include it in my portfolio as reference to my how far I have progressed since beggining my developer career. Real Time Bus Tracker

The Real Time Bus Tracker is my most resent project. This program displays a map of boston and shows the bus stops between MIT and Harvard one at a time, on a three second interval. This prosses begins only when the start button is pushed. I added my own personal touch to this program by styling the background and start button.
